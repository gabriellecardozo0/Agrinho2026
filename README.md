## 🎯 Sobre o Projeto

O **AgroFuturo Digital** é uma plataforma web criada com o objetivo de demonstrar como a tecnologia de ponta e a programação podem caminhar de mãos dadas com a preservação ambiental no campo. 

O projeto adota uma estética futurista (*Cyber-Agro*) para atrair a atenção dos jovens e mostrar que o agronegócio moderno é tecnológico, eficiente e focado em um amanhã sustentável.

### 🕹️ Funcionalidades e Interatividade
O site conta com o minigame **"Missão Irrigação Sustentável"**, desenvolvido em JavaScript. O jogo desafia o usuário a aplicar a quantidade exata de água em uma cultura agrícola. 
* **O aprendizado prático:** Se o jogador colocar pouca água, a planta não cresce; se colocar água demais, o sistema aciona um alerta de desperdício e zera o progresso. Isso simula o papel dos sensores de IoT (Internet das Coisas) reais na agricultura de precisão.

## 🛠️ Tecnologias Utilizadas

Para a construção deste projeto, foram utilizadas ferramentas reais de desenvolvimento de software:

* **HTML5:** Estruturação semântica do conteúdo.
* **CSS3:** Estilização futurista com uso de variáveis de cor, efeitos neon e design responsivo em estilo *Glassmorphism* (efeito de vidro).
* **JavaScript (ES6):** Lógica do jogo interativo e manipulação de elementos da página em tempo real.
* **GitHub & GitHub Pages:** Controle de versão do código e hospedagem pública do site.
